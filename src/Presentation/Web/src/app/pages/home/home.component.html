Documentação do Sistema

1. Introdução

Sistema completo para gestão de livros, autores e assuntos desenvolvido com Angular e Bootstrap.

Versão: 1.0.0
Última atualização: 29/03/2025

2. Funcionalidades

CRUD Completo
  • Gestão de Livros
  • Controle de Autores
  • Administração de Assuntos
Recursos Avançados
  • Relatórios em PDF
  • Validação de Formulários
  • Interface Responsiva

3. Tecnologias

Frontend
  • Angular 19+
  • Bootstrap 5
  • TypeScript
Backend
  • .NET 8
  • Entity Framework
  • SQL Server
Ferramentas
  • VS Code
  • Postman
  • Git
